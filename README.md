# LiftPointDB v15

Aggiornamento principale:
- AL55 con due configurazioni realmente selezionabili:
  - Simmetrica
  - Asimmetrica

Quando cambi configurazione:
- cambia la posizione dei perni bracci;
- cambia la geometria del simulatore;
- cambia il preset di posizione veicolo;
- cambia il calcolo delle distanze richieste.

Fonti:
- AL55 manuale: indice con layout simmetrico e asimmetrico; pagine 10 e 11.
- Il manuale indica due modi di installazione della traversa: simmetrico e asimmetrico.
- I dati veicolo restano template e devono essere sostituiti con dati ufficiali del costruttore veicolo.
